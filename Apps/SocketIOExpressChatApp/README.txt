Instructions for running the Socket.IO Chatroom Express App

Step 1: Open three command prompts
Step 2: In the first command prompt, run mongod
Step 3: In the second command prompt, run mongo
Step 4: In the command prompt from step 3, run use my_db 
Step 5: In the third command prompt, navigate to chat app directory
Step 6: In the command prompt from step 5, run npm install
Step 7: Run the app with nodemon index.js or node index.js


To test the chatroom with multiple users, open different tabs, windows, and/or browsers and 
visit localhost:3000/ in each of them.
