Running EmbeddedMApp

Step 1: Open 2 command prompts
Step 2: Navigate to the the embeddedMApp directory 
Step 3: run command npm install
Step 4: make sure you have a database "embeddedmap-development" db is in use!
Step 5: In other command prompt window, run mongod
Step 6: In command window where npm install was run, run nodemon app.js or node app.js
Step 7: Open web browser and navigate go to localhost:3000
